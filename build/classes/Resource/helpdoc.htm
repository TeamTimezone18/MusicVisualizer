JavaFX Audio Player

User Manual

 

Overview

 

            This application is a high-level interface for digital audio data where local MP3 and WAV files can be played from a user-configured playlist.

When a file is played, playback of the file’s audio data begins and the file’s available metadata is displayed. When the end of a file is reached, a new file from the playlist will start playing automatically.  Using the graphical controls or keyboard shortcuts, playback can be paused, the next or previous track from the playlist can be started, the playlist operation mode can be changed, and the volume and current playback position can be manipulated.

 

            To get started, first familiarize yourself with the major components of the graphical user interface:

 

 

           

            File Explorer                                                    -           Interface for browsing for audio files on the local system

            Playlist Tracks                                                  -           List of audio files to be played by the app

            Playlist Editing Controls                                  -           Buttons for manipulating the presence and order tracks in the playlist

            Playlist Operation Controls                             -           Buttons for controlling the tracks that are played from the playlist

            Playback Time and Volume Controls              -           Controls for manipulating the playback properties of the currently-playing track

           

 

Using the system

           

 

1.      Browsing for Files

                        The first step in using the program is to find the supported audio files on your system that you would like to play/view. To do so, the components of the file explorer interface are used as follows:

 

                                    Current File Explorer Path                               -           This is the directory that the file explorer is showing the contents of. Parent directories in the path (e.g. ‘Personal\’ in the image below) can be double-clicked to have the file explorer go up to that directory.

                                    Subfolder(s) of Current Path                           -           These are the child directories of the current file explorer directory. Child directories can be double clicked to have the file explorer go into that directory.            

Supported Audio Files at Current Path           -           These are the MP3 and WAV files that are in the current file explorer directory. Click different files while holding the CONTROL key to make multiple selections. These selections can be added to the playlist – see next section.

                                   

Note: When the app is opened, the file explorer will show the \Music\ directory of your user directory on the local machine. If that directory is unreachable, the file explorer will show the directory of the running application.

 

                       

 

           

 

2.      Building a Playlist

After the audio files of interest have been located in the file explorer, the next step is to add them to a playlist and edit the playlist as desired.  The controls to do so are as follows:

           

Add Tracks to Playlist                          -                       When this button is pressed, 1 or more selected files in the file explorer will be added to the end of the playlist. Files can also be added to the playlist by pressing ENTER while the file explorer is focused with selection, or by dragging 1 or more files from the file explorer and dropping in the playlist. Attempts to add directories or files that are already in the playlist are ignored.

                        Remove Tracks from Playlist              -                       When this button is pressed, 1 or more selected files in the playlist are removed from the playlist. Files can be deleted while they are playing and they will continue to play until a new track is loaded.

                        Move Track Down in List                    -                       When this button is pressed, a selected track is moved down one position in the playlist by swapping positions with the element below it. If multiple selections are made, only the most recently selected item is moved.

                        Move Track Up in List                         -                       When this button is pressed, a selected track is moved up one position in the playlist by swapping positions with the element above it. If multiple selections are made, only the most recently selected item is moved.

 

 

                                   

 

 

 

                                   

3.      Operating player

 

                  When one or more tracks are in the playlist, playback can be started and the player can be operated using the provided controls. To start the playback of a track at any time, double-click it in the playlist or press the ENTER key when it is selected in the playlist. Afterwards, the playback and playlist operation can be manipulated and viewed with the following controls:

 

Play/Pause                              -           This button toggles playback between paused and playing. Pause can also be toggled on/off by pressing the SPACE bar key.

Next Track                               -           This button starts the playback of a new track from the playlist, according to the playlist mode. In normal mode, the new track is the next in sequence with the previous. The next track can also be started by pressing the END key.

Previous Track                        -           This button starts the playback of a track that was previously played from the playlist, if it has not been deleted from the playlist. A previous track can also be started by pressing the BACKSPACE key.

Shuffle Mode                          -           This button sets the playlist mode to shuffle. When the playlist is in shuffle mode, skipping to the next track starts a random track from the playlist. When shuffle mode is ON, this button will have an orange border.

Repeat Mode                          -           This button sets the playlist mode to repeat. When the playlist is in repeat mode, skipping to the next track restarts the current track (if it is still in the playlist). When shuffle mode is OFF, this button will have an orange border.

Current Playback Time           -           This label indicates the current time of playback in correspondence with the Current Playback Progress slider.

Current Playback Progress      -           This slider displays and controls the current time of playback, beginning on the left and ending on the right. Click and drag the slider thumb to start playback from a new time. Alternatively, the LEFT and RIGHT keys can be used to adjust the current playback time backwards and forwards, respectively.

Current Track Duration           -           This label shows the total duration of the currently-playing track. This duration corresponds to the full right position of the progress slider.

Volume Slider                         -           This slider adjusts and displays the playback volume, with 100% system volume at the top of the slider and mute at the bottom. Click and drag the slider thumb to set a new volume. Alternatively, the F1 and F2 keys can be used to turn the volume down and up respectively.

 

                       

 

 

        

 

4.      Window sizing

The heights of the file explorer and playlist can be easily modified by clicking between them (i.e. the base of the file explorer or top of the playlist) and dragging up or down until the desired height is reached. The widths of the file explorer and playlist can be adjusted by clicking and dragging the left or right edge of the application window to resize the window. If either the file explorer or playlist are not showing in the application, it is likely that one has been resized to cover the other; resizing the application window or the playlist height will reveal the ‘missing’ component.

 

When the file explorer or playlist are not is use, they can easily be hidden to reduce the size of the application window. Click and hold the lower-right corner of the window, then drag the mouse towards the upper-left corner of the window cannot be made any smaller. To show the file explorer and playlist again, click and drag any edge of the application window to make the window larger.
